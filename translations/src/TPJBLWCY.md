# 《星战:旧共和国武士》中的爱之哲学

在任何讨论艺术的社区花上一段时间之后 任何人都会
开始拥有自己最喜欢的角色

《星球大战》也不例外

一个出自《星战：旧共和国武士》的绝地

不幸的是
